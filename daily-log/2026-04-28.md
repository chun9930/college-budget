# 오늘 한 일
- `AGENTS.md`에 이 프로젝트의 개발 방식 규칙을 추가했다.
- React + Vite SPA 구조에 맞춰 `src/pages`, `src/components`, `src/lib` 기반의 파일 구조를 만들었다.
- 홈, 예산 설정, 지출 기록, 분석 화면과 공통 헤더/하단 내비게이션을 구현했다.
- `localStorage` 저장 로직, 하루 예산 계산 로직, 경고 상태 판별 로직, 정기지출 반영 로직을 `src/lib`에 분리했다.
- `README.md`를 현재 구현 기준에 맞춰 수정했다.
- 예산 계산, 경고 상태, 정기지출 중복 방지 로직은 노드에서 직접 검증했다.
- `npm install`은 성공했지만, Vite 기반 `npm run test`와 `npm run build`는 현재 환경의 `spawn EPERM` 제한으로 완료하지 못했다.
