# 대학생 지출 관리 UI 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 대학생이 오늘 소비 가능 여부를 즉시 판단할 수 있는 미니멀 지출 관리 웹 UI를 만든다.

**Architecture:** 화면은 홈, 예산 설정, 지출 기록, 분석으로 분리한다. 경고 센터는 초기에는 별도 페이지가 아니라 홈의 경고 배너로 처리한다. 공통 상태는 `localStorage` 기반으로 관리하고, 계산 로직은 `lib`에서 분리한다. 모바일에서는 상단 요약과 큰 버튼을 우선 배치해 정보량을 줄인다.

**Tech Stack:** React, Vite, JavaScript, HTML/CSS, localStorage
