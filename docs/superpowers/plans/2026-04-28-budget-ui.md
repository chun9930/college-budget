# 대학생 소비 판단 UI 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 대학생이 지금 소비를 해도 되는지 즉시 판단할 수 있는 React + Vite 기반 소비 판단 UI를 구현한다.

**Architecture:** Hash Router 기반 SPA로 구성하고, 화면은 `pages`, 재사용 UI는 `components`, 계산과 저장은 `lib`로 분리한다. 로그인과 회원가입은 서버 인증이 아닌 localStorage 기반 mock auth로 처리한다. 홈 화면은 오늘 상태와 경고 배너에 집중하고, 예산 설정은 3개 카드 구조로 단순화한다.

**Tech Stack:** React, Vite, JavaScript, HTML/CSS, Hash Router, localStorage, Vitest, React Testing Library
