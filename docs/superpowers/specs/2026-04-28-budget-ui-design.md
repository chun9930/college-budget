# 대학생 지출 관리 UI 설계안

## 1. 목표

대학생이 지금 이 소비를 해도 되는지 빠르게 판단할 수 있는 미니멀한 지출 관리 UI를 설계한다.

핵심 원칙은 다음과 같다.

- 소비 기록보다 소비 판단을 우선한다.
- 화면마다 목적을 하나만 둔다.
- 모바일에서도 정보가 복잡해 보이지 않도록 구성한다.
- 로고는 교체 가능해야 하며, 크기가 바뀌어도 레이아웃이 깨지지 않아야 한다.

## 2. 화면 구성안

### 2.1 홈

홈은 오늘 상태를 한눈에 보는 화면이다.

포함 요소:

- 오늘 사용 가능 금액
- 오늘 지출 금액
- 상태 메시지
- 지출 입력 버튼
- 수입 / 예산 설정 버튼
- 고정지출 요약
- 경고 배너 또는 초과 배너

### 2.2 예산 설정

예산 설정은 월 수입과 하루 예산 기준을 관리하는 화면이다.

포함 요소:

- 월 수입 입력
- 자동 하루 예산 계산 결과
- 수동 예산 수정
- 남은 금액 이월 옵션
- 목표 저축 입력
- 비상금 제외 금액 입력
- 고급 설정 접기 영역

고급 설정:

- 수동 예산
- 이월 옵션

### 2.3 지출 기록

지출 기록은 실제 소비를 빠르게 입력하는 화면이다.

MVP 입력:

- 금액
- 카테고리
- 결제수단

고급 옵션:

- 지출유형
- 메모
- 템플릿 저장 / 불러오기
- 고정지출 / 정기결제 등록

### 2.4 분석

분석 화면은 소비 패턴을 단순하게 요약하는 화면이다.

포함 요소:

- 월 지출 요약
- 카테고리별 지출 비중
- 가장 많이 쓴 항목
- 과소비 카테고리
- 간단한 절약 힌트

## 3. 레이아웃 원칙

- 상단에는 공통 헤더를 둔다.
- 헤더에는 로고, 현재 페이지명, 주요 이동 버튼만 둔다.
- 본문은 상단 요약 카드, 핵심 입력 영역, 보조 정보 순서로 배치한다.
- 모바일에서는 카드를 세로로 쌓고, 버튼은 크고 짧게 표시한다.
- 색상만으로 상태를 전달하지 않고 텍스트도 함께 보여준다.
- 버튼은 최소 48px 높이를 확보한다.
- 본문 텍스트는 최소 16px 이상으로 유지한다.
- label과 focus 상태를 유지한다.

## 4. 실제 UI 와이어프레임 초안

### 4.1 데스크톱 홈

```text
┌──────────────────────────────────────────────────────────────┐
│ LOGO        오늘 현황                         예산 / 기록 / 분석 │
├──────────────────────────────────────────────────────────────┤
│ [오늘 사용 가능 금액]  [오늘 지출 금액]  [상태: 주의]          │
│ [경고 배너: 오늘 예산의 80% 이상 사용 중]                    │
│                                                              │
│ ┌─────────────────────┐  ┌─────────────────────┐             │
│ │ 지출 입력           │  │ 수입 / 예산 설정    │             │
│ │ 빠른 입력 버튼      │  │ 월 수입, 수동 예산  │             │
│ └─────────────────────┘  └─────────────────────┘             │
│                                                              │
│ [고정지출 요약] [정기결제 예정] [오늘 메모]                  │
└──────────────────────────────────────────────────────────────┘
```

### 4.2 모바일 홈

```text
┌──────────────────────┐
│ LOGO         메뉴    │
├──────────────────────┤
│ 오늘 사용 가능 금액   │
│ 24,500원              │
│ 상태: 주의            │
├──────────────────────┤
│ 오늘 지출 금액        │
│ 7,500원               │

│ [지출 입력]           │
│ [예산 설정]           │
├──────────────────────┤
│ 고정지출 요약         │
│ 월세 / 구독 / 교통    │
├──────────────────────┤
│ 경고 배너             │
│ 오늘 예산의 80% 이상 │
└──────────────────────┘
```

### 4.3 지출 기록

```text
┌──────────────────────────────────────────────┐
│ 지출 기록                                   │
├──────────────────────────────────────────────┤
│ 금액        [        ]                      │
│ 카테고리    [ 식비 ▼ ]                      │
│ 결제수단    [ 카드 ▼ ]                      │
│                                              │
│ [저장] [템플릿] [정기지출]                   │
└──────────────────────────────────────────────┘
```

## 5. 상태 표현

- 안전: 초록 계열
- 주의: 파란 계열 또는 중간 강조색
- 위험 / 초과: 강한 붉은 계열

상태는 색상만으로 표현하지 않고 반드시 텍스트를 함께 표시한다.

## 6. 비고

- 대학생 할인 정보는 초기 UI에서 제외한다.
- 분석은 복잡한 추천보다 단순한 요약 중심으로 시작한다.
- 로그인은 핵심 UI가 안정된 뒤 보조 화면으로 다룬다.
- 경고 센터는 별도 1차 화면이 아니라 홈 배너로 시작한다.
